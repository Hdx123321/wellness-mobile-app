# Demo Script

Target duration: no more than 15 minutes.

1. Show the architecture and explain Kotlin, Spring Boot, MySQL, and backend-only AI access.
2. Register and log in as a new user.
3. Create, view, update, and delete a wellness record.
4. Show the Dashboard and seven-day summary.
5. Ask the AI assistant a question grounded in recent records.
6. Restart the app and show persisted records and chat history.
7. Generate a recommendation if the optional agent workflow is implemented.
8. Log out and verify protected screens are unavailable.
9. Briefly show authorship and test results.
